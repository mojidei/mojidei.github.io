因为要开始写软工课用的代码了所以就打算整一下idea，因为eclipse有点丑。

先配一下maven（一个构建工具）用idea默认自带的maven就行，maven仓库也用默认的

配一下镜像，参考https://developer.aliyun.com/mvn/guide

文件路径：C:\Program Files\JetBrains\IntelliJ IDEA Community Edition 2023.2.5\plugins\maven\lib\maven3\conf\settings.xml

在<mirrors></mirrors>标签中添加 mirror 子节点

配置完镜像需要重启idea

接下来新建项目，使用课本上说的最简单的方法……从网上下模板，然而spring官方不提供java8版本的模板了，所以我们从阿里这块下：https://start.aliyun.com/

选俩依赖工具：Spring Boot DevTools、Spring Web

解压好后，idea 新建->来自现有源代码的项目

项目建好惹，找到DemApplication这个文件，点开后再点右上角小三角运行

浏览器http://127.0.0.1:8080/ ，一个helloworld页面就展现在眼前了……

然而我并没有配置tomcat，怎么回事捏

另外收藏一下这个https://mvnrepository.com/ ，配pom文件用的
