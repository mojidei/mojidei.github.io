创建项目

为了避免混乱，我们在D盘新建了名为vueproject的文件夹作为所有vue项目的上级文件夹。我们以日期命名项目名，并且只用数字命名。

以命令行方式创建项目：

用文件资源管理器打开vueproject文件夹，在地址栏输入cmd，回车，呼出工作目录为D:\vueproject的cmd命令行窗口。
创建项目命令：
vue create 0619
0619是项目名
之后：

  1. 选择manuslly手动配置（上下键切换，enter确定）

  2. 仅仅选中babel、router、vuex、css pre-processors（空格键选中/取消选中）

  3. 选2.x（这是vue版本）

  4. 一直回车

  5. 进入项目路径。命令：

    cd 0619

  6. 运行项目。命令：

    npm run serve

  7. 访问本地页面：在主流浏览器地址栏输入

    localhost:8080

回车

要停止运行，可使用ctrl+c
