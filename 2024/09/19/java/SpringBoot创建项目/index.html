SpringBoot创建项目

mojidei
2024-09-19

https://start.spring.io/

Project栏让你选构建工具吧，我是maven

Language语言，我是Java

Spring Boot版本 看情况选

Project Metadata 类名之类，有自己的规范；Packaging打包方式，我选Jar；Java版本看情况

Dependencies包，里面有各种各样工具，一般Spring web和SpringBootDevTools这俩必选。前者是构建Web必须，后者用来帮助开发。

选好后就Ctrl+Enter把压缩包下载下来

解压好后用idea打开，方式是双击pom文件

然后等右下角读条读完
