新学期，新挑战

今天老师提了一下java设计模式要做期末项目，我呢也有了一个脑洞小想法，准备小干一场。

既然科目名称是java设计模式，我打算用java而不是其他语言以避免不必要的麻烦。

由于上个学期学servlet和spring的时候感觉非常费劲，我打算用个轻量好用的框架……于是开始谷歌java后端框架排行。

首先映入眼帘的是 https://javalin.io/ ，然而似乎它没有中文文档（或许有，我没找），然后看到了 https://lets-blade.github.io/ 。

另外使用时也可以参考 blade/README_CN.md at v2.1.3 · lets-blade/blade · GitHub

基于java8，听上去很符合中国国情，因为据说业界主流是java8。拉下来看下面有数据库操作库 和 JSON 扩展 ，据说”任何自定义应用程序开发人员都可以在一天内快速理解这个框架”。就你了。
